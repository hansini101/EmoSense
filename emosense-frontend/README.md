# EmoSense Frontend

AI-powered emotional wellness platform for university students. Detect emotions, track mood patterns, chat with AI therapist Luma, and access mental health resources.

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

You can start editing pages by modifying files in the `app/` directory. Changes update automatically as you edit.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components (UI library + custom)
- `lib/` - Utility functions and helpers
- `styles/` - Global and component styles
- `hooks/` - Custom React hooks
- `public/` - Static assets

## Key Features

- **Emotion Detection**: AI-powered facial emotion recognition
- **Mood Tracking**: Visual charts and mood history
- **Luma**: AI wellness companion for support
- **Wellness Hub**: Breathing exercises, gratitude journal, playlists
- **Resource Directory**: Campus counselors and mental health services

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Shadcn/UI components
- Radix UI primitives
